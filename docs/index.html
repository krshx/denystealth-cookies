1. What NoConsent Does

NoConsent is a Chrome browser extension that automatically denies non-essential cookie consents and closes consent banners on websites you visit. It operates entirely within your browser and does not communicate with any external server.

2. Data We Do NOT Collect

  • We do not collect your browsing history or the URLs of sites you visit
  • We do not collect personal identifiers (name, email, IP address, device ID)
  • We do not use cookies, pixels, or any tracking technology
  • We do not store any information about which websites you use the extension on
  • We do not sell, share, or trade any data with third parties

3. Optional Anonymous Telemetry Opt-in only

NoConsent includes an entirely optional telemetry feature that you must explicitly opt in to. If you choose to enable it, the extension sends a single anonymous ping when a denial is performed, containing only:

  • A randomly generated session ID (not tied to you or your device, regenerated each session)
  • The type of CMP detected (e.g. "OneTrust", "Cookiebot") â€” no domain or URL
  • The count of consents denied (e.g. "847")
  • The extension version number

This data contains no personally identifiable information. It is used solely to improve the extension by understanding which CMPs are most common. You can disable this at any time in the extension settings. Telemetry is off by default.

If telemetry is enabled, anonymous pings are sent to a Cloudflare Worker endpoint and stored in an aggregate-only database. No individual records are retained beyond 24 hours.

5. Local Storage Only

Your opt-in/opt-out preferences are stored using Chrome's storage.local API, which keeps data on your device only. This data is never transmitted anywhere.

6. Children's Privacy

NoConsent does not knowingly collect data from anyone, including children under 13. The extension has no age-gated features.
